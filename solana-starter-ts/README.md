# Solana Starter TS

## Setup

To run the scripts in this folder you need to make sure to have a `wba-wallet.json` file in the `wallet` folder.

## Running the scripts

To install the project run:

```bash
yarn
```

To execute the scripts run:

```bash
yarn <script-name>
```

Available scripts see in the following.
